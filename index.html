주기성과 추세를 지닌 비정형 시계열 데이터의 미래 값을 예측하는 모델 설계

Stationarity tests, seasonal decomposition, ARIMA modeling, residual diagnostics

💡 주요 인사이트 | Key Insights
AIC 기반 최적 모델 도출, 계절성 보정의 중요성 확인
Identified ARIMA(1,1,1) as optimal; highlighted need for seasonal correction
